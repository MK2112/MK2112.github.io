Any_To_Any.py

November 2023 - Preprocessing Utils

The ability to swiftly convert media files between different file formats poses a critical necessity in data processing. With Any_To_Any.py, I wanted to address this need through providing a Python-based, lightweight tool that allows to switch between an array of media formats. From audio files like MP3 and FLAC, to image-based formats like PNG, GIF, BMP, and video files like MP4, WebM, and AVI.

In short, this tool aims to just solve the headache that can be file conversion, all with a straightforward interface.
