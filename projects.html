January 2024 • NLP

GPTemplate

GPTemplate is a fully trainable and usable Generative Pre-Trained Transformer (GPT) within a Jupyter Notebook. I built this as a progression beyond Andrej Karpathy's Neural Networks Series to showcase GPT's implementation efficiency and basic inner workings. Users only need to provide a text file. GPT internalizes the textual patterns therein and generates contextually aligned text, given a prompt.

November 2023 • Preprocessing Utils

Any_To_Any.py

The ability to swiftly convert media files between different file formats poses a critical necessity in data processing. Any_To_Any.py is a Python-based, lightweight tool that allows conversion between an array of media formats, such as audio, image, and video files. This tool aims to solve the headache of file conversion with a straightforward interface.
