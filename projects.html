MiniCorpus

February 2025 • NLP

MiniCorpus investigates, reproduces and improves MiniPile, a distilled subset of The Pile Deduplicated. Introducing several sampling techniques, like size-density-proportionate sampling, MiniCorpus creates training datasets that are smaller than MiniPile while maintaining competitive performance. The project provides a complete pipeline for dataset distillation, from embedding generation to cluster-based sampling, with comprehensive benchmarking across multiple sampling strategies and sketches theoretical ideas too.

MobileYOLOv3

October 2024 • Vision

MobileYOLOv3 is a tiny, back-to-back implementation of the YOLOv3 object detection algorithm in PyTorch. Its aim is to showcase an entire training pipeline, from data preparation to model inference, in a single Jupyter Notebook. The model is trained on the ICDAR2015 dataset and gets one-shot pruned and quantized, making it suitable for use cases like real-time text detection on resource-constrained devices.

GPTemplate

January 2024 • NLP

GPTemplate is a fully trainable and usable Generative Pre-Trained Transformer (GPT) within a Jupyter Notebook. I built this as a progression beyond Andrej Karpathy's Neural Networks Series to showcase GPT's implementation efficiency and basic inner workings. Users only need to provide a text file. GPT internalizes the textual patterns therein and generates contextually aligned text, given a prompt. The hugely popular GPT approach to NLP is thus made a bit more accessible.

Any_To_Any.py

November 2023 • Preprocessing

The ability to swiftly convert media files between different file formats poses a critical necessity in data processing. With Any_To_Any.py, I wanted to address this need through providing a Python-based, lightweight tool that allows conversion between an array of media formats. From audio files like MP3 and FLAC, to image-based formats like PNG, GIF, BMP, and video files like MP4, WebM, and AVI.

In short, this tool aims to just solve the headache that can be file conversion, all with a straightforward interface.
