Long Short-Term Memory

February 2024 - Reading Time: 5 Minutes

Long Short-Term Memory is a recurrent neural network type for learning long-term dependencies. This was hard to achieve, especially due to the vanishing gradient problem experienced up until that point. LSTM was the first in overcoming this issue, and it proved foundational for many subsequent advances in deep learning. Here's why it works.
